# Harshith-codealpha-Projects
Task - 2 : Stock Portfolio Tracker 


A simple Python-based stock portfolio tracker that calculates the total investment value based on user-input stock names and quantities using predefined stock prices.

This project is created as part of an internship task to demonstrate basic Python concepts like dictionaries, input/output, arithmetic operations, and file handling.


Features of the Stock Portfolio Tracker :

1.Add multiple stocks with quantities

2.Uses predefined (hardcoded) stock prices

3.Calculates total investment value

4.Option to save results to .txt or .csv

5.Beginner-friendly and easy to modify
