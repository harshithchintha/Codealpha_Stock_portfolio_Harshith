# Harshith-codealpha-Projects
Task - 2 : Stock Portfolio Tracker 


A Python based stock portfolio tracker that calculates the total investment value based on user-input stock names and quantities using predefined stock prices.

This project is created as part of an internship task to demonstrate basic Python concepts like dictionaries, input/output, arithmetic operations, and file handling.


Features of the Stock Portfolio Tracker :

1.Add multiple stocks with quantities

2.Uses predefined (hardcoded) stock prices

3.Calculates total investment value

4.Option to save results to .txt or .csv

5.Beginner-friendly and easy to modify

How to Run :

Install Python (3.10)

Clone the repository:

git clone https://github.com/harshithchintha/stock-portfolio-tracker.git


Navigate to the folder:

cd stock-portfolio-tracker


Run the script:

python stock_tracker.py

Example Usage :

How many stocks? 2
Stock name: AAPL
Quantity: 3
Stock name: TSLA
Quantity: 2

Total Investment = 1040
Save to file?  yes
txt or csv?  csv
